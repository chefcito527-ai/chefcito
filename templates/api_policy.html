Política de Uso de API Keys

Última actualización: Enero 2026

01

¿Qué es una API Key?

Una API Key (clave de API) es un token de autenticación que le permite acceder a servicios de terceros, en este caso la IA de Google Gemini. Es similar a una contraseña que identifica al solicitante ante el proveedor del servicio.

Chefcito requiere que usted aporte su propia API Key de Google Gemini para poder generar recetas. Esto significa que usted controla el acceso directamente desde Google.

02

Seguridad y Uso de su API Key

Su API Key se utiliza exclusivamente para realizar solicitudes a la API de Google Gemini en su nombre, enviando los ingredientes o el prompt. La clave se envía cifrada desde el cliente.

🔒

Su API Key nunca se almacena en nuestra base de datos. Ni la guardamos, ni la analizamos ni hacemos peticiones en "segundo plano" sin que usted active el botón deliberadamente.

Con su clave NO:

  • La guardamos en ninguna base de datos, logs, cookies o archivo.
  • La usamos para hacer peticiones automatizadas sin su acción directa.

Por tanto, el uso es completamente de su responsabilidad, manténgala confidencial y vigile su consumo personal en la consola de Google.

03

Revocar API y Contacto

Si cree que su clave ha sido filtrada externa a esta web, simplemente iníciela en Google y cancélela o revoque. Chefcito no tiene manera manual ni forma de ayudar en filtraciones debido a que solo existe en su navegador.
